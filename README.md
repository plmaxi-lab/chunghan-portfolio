# chunghan.work — portfolio

Self-contained static site: homepage plus four case studies.

## Files
- `index.html` — the entire site (homepage + case-study reader)
- `assets/` — case study screenshots

## Run locally
Open `index.html`, or run `python3 -m http.server` in this folder.

## Deploy to Vercel
1. Push this repo to GitHub.
2. In Vercel: Add New Project, import the repo (framework preset: Other).
3. Add `chunghan.work` as a custom domain under Settings -> Domains; point DNS there.
